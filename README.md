# nvim
This is my neovim setup based of NvChad
1) Install Neovim
2) Install NvChad with the following command
     `git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1 && nvim`
3) Say No to the first prompt that opens when you download it and allow everything to install and download
4) Go to `~/.config` and replace your `nvim` file with mine
5) use :MasonInstallAll to install the lsps I have, right now I believe I only have one for python
## NOTE:
If you use this, make sure you change the clipboard as I'm using WSL so I have a line for it in `/lua/custom/init.lua`
